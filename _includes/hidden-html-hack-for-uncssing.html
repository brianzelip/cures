This file is only here for purposes of having the three classes present in original html. Each of these three classes are dynamically generated by javascript on page load, and therefore aren't present in original html. The build process gulp-uncss, which removes all CSS rules that aren't used in any html files that call the CSS file, removes these three classes from the final CSS file because they aren't found in any original html files. Creating this hack of a fucking file allows us to not hack up our original html files!
